# Smart Hostel / PG Management System

This project automates hostel/PG management using:

- QR + FaceID based attendance
- Smart complaint tracking
- Room allocation module
- Visitor logging system
- IoT-based power & water monitoring

## Project Structure
backend/   → API server (FastAPI/Node)  
frontend/  → Web dashboard (React)  
iot/       → Sensor firmware & MQTT scripts  
docs/      → ER diagrams, architecture, Jira references  

## Tech Stack
- FastAPI / Node.js  
- React.js  
- PostgreSQL  
- ESP32 + MQTT
